# Project setup

```bash

$  npm install

```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## APIs

### Authorizations

/auth/login, POST
/auth/register, POST

### Search for Users

/users/search, GET

### APIs for friendRequests

/friends/request, POST
/friends/requests, GET
/friends/respond, PUT
